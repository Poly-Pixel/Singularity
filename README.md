# Singularity

A free, easy to use Discord bot

## What features does it have?

**Singularity** has the following features:

- Customizable Welcome & Leave Messages
- Customizable Reaction Roles
- Moderation Commands
  - Ban
  - Kick
  - Timeout
  - Unmute
  - Clear (bulk deletes messages)
  - Warn
  - Nicknaming
  - View infractions
- Polls
- Stats
- My Singularity
- ...and more!
